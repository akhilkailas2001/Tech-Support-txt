git commands:

git init	----for starting
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
git add --all	----add into git
git commit -m "add comments"	----add into master
git remote add remotename repositorylink    ----for linking github with git
git remote -v  ---find which remote we are using
git push remotename master   -----to save project to github
